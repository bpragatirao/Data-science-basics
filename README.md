# 📊 Data Science Basics: Learning & Implementation Lab

Welcome to my Data Science journey! This repository is a structured collection of my self-study progress, covering everything from raw data manipulation in **SQL** to building predictive models in **Python**.

---

## 🗺️ Learning Roadmap
My goal is to master the end-to-end Machine Learning pipeline. I am currently following this phase-wise approach:

### Phase 1: Data Engineering & Exploration
* **SQL Mastery:** Joins, CTEs, Window Functions, and Data Normalization.
* **Exploratory Data Analysis (EDA):** Handling missing values, Outlier detection, and Data Profiling.
* **Visualization:** Creating storytelling dashboards using Matplotlib, Seaborn, and Plotly.

### Phase 2: Statistical Modeling & ML
* **Supervised Learning:** Regression (Linear/Polynomial) and Classification (Logistic, KNN, SVM, Trees).
* **Unsupervised Learning:** Clustering (K-Means) and Dimensionality Reduction (PCA).
* **Mathematics:** Linear Algebra, Calculus, and Inferential Statistics.

### Phase 3: Advanced AI & Deployment
* **Deep Learning:** Basics of Neural Networks and CNNs.
* **NLP:** Text processing, Sentiment Analysis, and Word Embeddings.
* **Model Deployment:** Understanding how to wrap models in APIs (Flask/FastAPI).
