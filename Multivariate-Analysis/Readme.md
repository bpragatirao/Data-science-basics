# Titanic Dataset: Multivariate Analysis

This project focuses on **Multivariate Analysis**, examining how multiple features interact with each other and the target variable (`Survived`).

## Key Insights

* **Survival by Class and Gender**: Heatmaps and Barplots show that females in 1st and 2nd class had the highest survival rates.
* **Age vs. Survival by Class**: Boxplots reveal that 1st-class passengers were generally older, while 3rd-class passengers were younger with a narrower age distribution.
* **Fare vs. Class**: Correlation plots confirm a strong relationship between higher fares and 1st-class tickets.
* **Pair Plots**: Scatter matrix plots were used to visualize the distribution of numerical features (Age, Fare, Pclass) colored by survival status to identify clusters.

## Visualizations Used
* **Heatmaps**: To visualize correlation matrices between all numerical features.
* **FacetGrids**: To compare distributions (like Age) across multiple categories (Sex and Pclass) simultaneously.
