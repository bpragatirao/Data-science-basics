# TMDB Movie Data Fetching & Visualization

This project demonstrates how to programmatically interact with the **The Movie Database (TMDB) API** to retrieve cinematic data and perform exploratory data analysis (EDA) using Python in a Jupyter Notebook environment.

## 📌 Project Overview
The goal of this project is to build a pipeline that:
1.  **Extracts** live data from the TMDB API (Top Rated, Popular, or Genre-specific).
2.  **Cleans** and structures the JSON response into a Pandas DataFrame.
3.  **Visualizes** trends such as budget vs. revenue, genre popularity, and rating distributions.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Data Handling:** `Pandas`, `NumPy`
* **API Requests:** `requests`
* **Visualization:** `Matplotlib`, `Seaborn`, `Plotly` (Optional for interactive charts)
* **Environment:** Jupyter Notebook

### Prerequisites
You will need a TMDB API Key. 
* Create an account at [themoviedb.org](https://www.themoviedb.org/).
* Go to your Settings > API to generate your **API Read Access Token** or **API Key**.
